Project Name: WorldSurf

Project Description

Have you ever desired to ride the best waves around the world? We do!

Over the next four weeks, mobile and web wireframes will be built for fictional company, WorldSurf. From a user experience design standpoint, we will create simple forms, easy navigation, attractive calls to action, and informative charts and tables for the best surfing experience on the planet. We invite the public to try each set of wireframes through our nav flow tests on UsabilityHub.
